# X Stopwatch (MM:SS)

A simple React stopwatch application that displays **minutes and seconds (MM:SS)**.  
Users can **start, pause, and reset** the timer.

---

## Features

- Start / Pause functionality
- Reset the stopwatch
- Display time in **minutes:seconds** format
- Built with React functional components and `useState` + `useEffect`
